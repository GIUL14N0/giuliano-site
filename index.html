Projetos

  • Algoritmo Genético para o Problema de Localização de Estações de Carregamento de Veículos Elétricos

    Meu trabalho de conclusão de curso abordou um tema crucial para o futuro da mobilidade elétrica. Formulei o problema de localização como um problema de P-medianas utilizando um algoritmo genético para identificar as localizações ideais das estações de carregamento. O algoritmo foi projetado para minimizar os custos de locomoção dos usuários de veículos elétricos até as estações de carregamento, apresentando soluções competitivas e eficazes para o planejamento urbano e a gestão energética.
